## **VIM Configuration**

### Installation:

```sh
cd ~
git clone git@github.com:degete/vimrc.git ~/.vim
ln -s .vim/vimrc .vimrc
# For NeoVIM
ln -s ~/.vim .config/nvim
vim +PlugInstall +qall now
```
